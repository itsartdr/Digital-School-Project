Get Started Learn More

We have courses for all levels of programming, from begginer level to back end development.

Begginer

We have simple courses that helps introduce small children into programming.

Intermediate

After a short introduction to programming, we teach you easy to learn programming languages on code.org and scratch, with lessons in Kodu, MIT App Inventor and wordpress!

Advanced

Now, we introduce you to slightly more difficult programming langauges such as CSS and JavaScript.

Expert

After learning JavaScript and CSS, we teach Java and back end development to simplify the work of making a game or a website.
